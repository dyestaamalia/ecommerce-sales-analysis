## E-Commerce Sales Analysis (Excel)

### Deskripsi

Project ini merupakan analisis data penjualan e-commerce menggunakan Microsoft Excel, dimulai dari tahap data cleaning hingga pembuatan dashboard interaktif. Tujuan utama dari project ini adalah untuk memahami performa bisnis melalui pengolahan data, visualisasi, serta identifikasi pola dan tren penjualan yang dapat mendukung pengambilan keputusan.

### Dataset

Dataset yang digunakan dalam project ini berasal dari:
Kaggle – E-Commerce Data
(https://www.kaggle.com/datasets/carrie1/ecommerce-data](https://www.kaggle.com/datasets/carrie1/ecommerce-data)

Dataset ini berisi data transaksi penjualan online, termasuk informasi seperti invoice, produk, quantity, harga, tanggal transaksi, dan negara.

### Data Preparation

Data diproses melalui beberapa tahap data cleaning, antara lain:

* Menghapus data kosong (missing values)
* Menghilangkan data duplikat
* Menyesuaikan tipe data pada kolom numerik seperti quantity dan unit price
* Menyiapkan data agar siap digunakan untuk analisis dan pembuatan pivot table

### Analysis Process

Data yang telah dibersihkan kemudian diolah menggunakan Pivot Table di Excel untuk menghasilkan ringkasan metrik utama. Proses ini mencakup:

* Agregasi total penjualan (sales)
* Perhitungan total profit
* Perhitungan jumlah transaksi
* Analisis jumlah produk terjual (quantity sold)
* Pengelompokan data berdasarkan waktu, produk, dan negara

### Dashboard Description

Dashboard yang dihasilkan menyajikan informasi dalam bentuk visual yang mudah dipahami, dengan beberapa komponen utama:

* Key Performance Indicators (KPI) yang menampilkan total sales, total profit, total transaksi, dan jumlah produk terjual
* Grafik tren penjualan bulanan untuk melihat pola perubahan penjualan dari waktu ke waktu
* Visualisasi penjualan berdasarkan negara untuk mengidentifikasi kontribusi geografis
* Daftar produk dengan profit tertinggi untuk mengetahui produk yang paling menguntungkan
* Fitur filter interaktif (tahun, bulan, dan negara) untuk memungkinkan eksplorasi data secara dinamis

### Key Metrics

* Total Sales: 435.354.334
* Total Profit: 87.070.867
* Total Transactions: 227.703 transaksi
* Quantity Sold: 2.394.679

### Insights

Hasil analisis menunjukkan beberapa temuan utama:

* Penjualan mengalami fluktuasi sepanjang periode waktu dengan puncak pada bulan tertentu
* Sebagian besar kontribusi penjualan berasal dari wilayah tertentu, khususnya United Kingdom
* Distribusi profit menunjukkan bahwa sebagian kecil produk memberikan kontribusi yang signifikan terhadap total keuntungan, yang mengindikasikan adanya pola Pareto
* Terdapat peluang untuk meningkatkan performa bisnis melalui ekspansi pasar dan optimalisasi produk unggulan
